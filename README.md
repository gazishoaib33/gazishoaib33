## Featured Projects

### Explainable ML for Mental Health Risk Prediction
> MSc Thesis · XGBoost · SHAP · LIME · Python

Designed an interpretable XGBoost classifier (**87% accuracy**) to identify key mental health risk factors among university students. Used SHAP and LIME to make predictions explainable to non-technical clinicians — optimised classification threshold to minimise false negatives in a high-stakes health context.

---

### Disaster Damage Classification via Satellite Imagery
> CNN · TensorFlow · OpenCV · Python

Built a deep learning pipeline to classify structural damage severity from satellite images, enabling faster post-disaster resource allocation. Achieved **82%+ accuracy** on an imbalanced multi-class dataset using augmentation and class-weight balancing.

---

### Loan Approval Prediction — End-to-End ML Pipeline
> Scikit-learn · Pandas · Power BI · Python

Full pipeline from raw data ingestion to a live Power BI dashboard achieving **85% prediction accuracy**. Identified the top 5 credit risk drivers via feature importance analysis — providing direct input for lending policy decisions.

---

### BizInsight AI — Business Intelligence Assistant *(Concept)*
> Python · NLP · LLM · Power BI API

AI-powered assistant that interprets natural-language queries against structured business data, returning plain-English summaries and auto-generated charts — making analytics accessible to non-technical decision makers.
